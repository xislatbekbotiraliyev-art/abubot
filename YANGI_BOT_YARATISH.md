# Yangi Bot Yaratish (1 daqiqa)

## 1. BotFather ga kiring
Telegram da `@BotFather` ni oching

## 2. Yangi bot yarating
```
/newbot
```

## 3. Bot nomini kiriting
Misol: `Kinomaniya Bot`

## 4. Username kiriting (bot bilan tugashi kerak)
Misol: `kinomaniya2024_bot`

## 5. Token ni nusxalang
BotFather sizga token beradi:
```
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

## 6. Local .env faylini yangilang
```
BOT_TOKEN=yangi_tokeningiz
ADMIN_IDS=5561294515
PORT=10000
```

## 7. Render da Environment Variables ni yangilang
- Render Dashboard → Settings → Environment
- `BOT_TOKEN` ni yangi token bilan almashtiring
- Save Changes

## 8. Tayyor!
Bot avtomatik qayta deploy bo'ladi va ishlaydi!

---

## Yoki eski botni tiklash (30 daqiqa kutish kerak)

Agar eski botni saqlamoqchi bo'lsangiz:
1. 30 daqiqa kuting
2. Telegram avtomatik logout holatini bekor qiladi
3. Keyin `python main.py` ishga tushadi

Lekin yangi bot yaratish tezroq! 🚀
